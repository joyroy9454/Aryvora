# Weekly Global Tech Updates — Editorial Workflow

## Publishing Schedule
Every Friday (or Monday), publish a new weekly digest covering the week's top tech and AI news.

## FileNaming
- `content/weekly/YYYY-MM-DD.md` — date should be the publication date
- Slug is auto-generated from filename

## Content Template
Use `hugo new weekly/YYYY-MM-DD.md` to create from archetype, or copy the previous week and update.

## Research Sources (Priority Order)
1. Official company blogs (blog.google, newsroom.ibm.com, etc.)
2. Hacker News top stories (score > 100)
3. Google News RSS feeds
4. Wikipedia for context on protocols/standards
5. GitHub trending for open source projects
6. Security advisories (CVE databases, vendor advisories)

## Scoring Criteria (rate each candidate 1-5)
- Traffic potential (search volume)
- Relevance to audience (students/developers)
- Usefulness (actionable advice)
- Freshness (published this week)
- Ease of explanation (simple English)
- Internal linking opportunity (connects to existing posts)

## Minimum Threshold
Only publish updates that score 3+ on traffic potential AND relevance.

## Internal Linking Rules
- Link to at least 3 evergreen guides from every weekly post
- Link to 2 related posts in the same category
- Use descriptive anchor text (not "click here")

## Pre-Publish Checklist
- [ ] Facts verified against primary sources
- [ ] No unsupported claims
- [ ] Title is unique and descriptive
- [ ] Meta description under 160 chars
- [ ] All internal links work
- [ ] Mobile layout tested
- [ ] No duplicate content from other posts
- [ ] FAQ section included if 3+ natural questions exist

## SEO Targets (use naturally, don't stuff)
- weekly tech updates
- AI news this week
- latest AI updates
- developer news
- tech digest
- what's new in tech
- weekly AI roundup
- AI tools update
- cybersecurity updates
