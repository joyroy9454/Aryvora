About Aryvora#

Aryvora is a digital publication creating honest AI tool reviews, practical tech guides, and premium eBooks for students, creators, and tech learners.

Founded and run by Joy Roy — a 2nd-semester BSc Data Science student who builds things with AI while studying, not after graduating.

Who I Am#

Hey, I’m Joy. I’m a BSc Data Science student who calls himself a “vibe coder” — someone who uses AI tools to build real things fast.

I started Aryvora because I was tired of reading vague, surface-level tech articles that never actually helped me do anything. So I built a system that creates content I’d actually want to read.

What I know:

  • C, Python, MySQL, Data Structures & Algorithms
  • Basic web development (HTML, CSS, JS)
  • AI tools and workflows (the real, practical kind)
  • Building and deploying websites with Hugo + GitHub Pages
  • Vibe coding with Cursor, Replit, and other AI-powered dev tools

What I’m learning:

  • Machine learning and deep learning
  • Cloud deployment and DevOps
  • How to run a real online business

My Philosophy#

  1. Practical over theoretical. Every product includes steps you can use today.
  2. Beginner-friendly. No jargon without explanation.
  3. Honest. I only recommend tools and methods I’ve actually tested.
  4. AI-assisted, not AI-dependent. I use AI to research, draft, and refine — but every piece is reviewed and improved by a human (me).

What Makes This Different#

Most “AI blogs” are just spammy listicles generated by bots and forgotten. This is different:

  • I’m a real student writing for real students
  • Every tool is tested before it makes it into an article
  • SEO-optimized so you can actually find what you’re looking for
  • Monetization-ready — this is a real business, not a hobby project
  • Full transparency — I use AI tools to help create content, and I’m upfront about it
